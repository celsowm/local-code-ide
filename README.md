## Overview

A VS Code-inspired IDE built with C++20 and Qt 6, featuring:

- 📁 File explorer with tree view
- ✨ Syntax highlighting (C++, and more)
- 🤖 Local AI assistant with tool calling
- 🔀 Split editor support
- 📊 Git integration
- 🎨 Modern dark theme

## 📸 Features

- **Explorer** - File tree with folder collapse
- **Editor** - Split view, syntax highlighting
- **Assistant** - Local AI with tool calling
- **Git** - Stage, commit, diff
- **Quick Open** - `Ctrl+P`
- **Command Palette** - `Ctrl+Shift+P`
